# NW Homecare Medical — Demo Site

Static demo site for **NW Homecare Medical** (nwhomecaremed.com).  
Certified Age In Place Specialists serving the Portland, OR metro area.

## Service Pages
- `/` — Homepage
- `/bath-remodel/` — Best Bath Accessible Bathroom Remodels
- `/overhead-lifts/` — Molift Ceiling Track Lift Systems
- `/wheelchair-ramps/` — EZ Access Modular Aluminum Ramps
- `/stairlifts/` — Harmar Straight Stairlifts

## Stack
Pure HTML + CSS. No build step. Deployed via Netlify.

## Deploy
Push to `main` → Netlify auto-deploys from the repo root.
